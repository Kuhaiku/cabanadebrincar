

a pagina chamada Painel, vai ser um index para todas as seguintes paginas:


não vai mais ter o financeiro, o painel vai virar uma pagina "pedidos" onde vão estar todos os pedidos e a parte de copiar o orçamento adicionar se tiver taxas e etc, ai depois de copiado o orçamento vai ter um botão para "orçamento aprovado enviar para agenda"

Pagina da Agenda, é uma pagina onde vai ser um grande calendário, onde os dias que tem reserva para festa vão estar listados e afins, aqui é onde vamos gerenciar as festas, ao clicar na reserva vamos poder ver TODOS os detalhes dela, e de dentro da agenda a gente vai poder lançar para "serviços concluidos" e nela vai ter o botão para gerar o link para avaliar tambem.

Pagina dos comentários: o cliente vai receber uma página com o link dele, cada um vai receber um link, vai poder escerver até 350 caractéres e vai poder colocar as estrelinhas de 1 a 5, as estrelas ja vem as 5 preenchidas por padrão

página serviços concluidos, aqui é onde vamos poder abrir e anotar os gastos que tivemos para cada uma das festas, e salvar.

pagina financeiro, vai carregar todos os dados de financeiro, vamos poder fazer anotações de despesas indiferente de pedidos tipo sei lá "gasto do Mês" a anotação vai consistir em um titulo campo de texto, um select para "entrada/saida" e um campo de valor para o numero

Pagina preços, vamos mudar essa pasta para "gerenciador do formulário"
Aqui é oque controla cada coisa do nosso formulário, com isso nós podemos manter a ideia de adicionar e remover itens do nosso formulário ou ocultar para caso de não ter os ites disponiveis.

Avaliações: nessa página vamos exibir todas as avaliações que os usuários fizeram, e nela vamos poder ocultar, excluir e editar, para caso tenham erros de grafia.
